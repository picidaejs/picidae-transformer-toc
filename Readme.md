## Options


| name | type | desc |
| --- | -- | ---- |
| maxDepth | number | max depth of toc, default `3` |
| test | string | matching text, case insensitive. default `"<TABLE_OF_CONTENTS>"` |
| force | boolean | whether force effect for all markdown, even don't match `test` |
